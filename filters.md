# Allowed filters in ARIADNE
**ARIADNE** uses pyphot (<https://mfouesneau.github.io/docs/pyphot/>) as an interface for photometry, and thus we use the same filter naming convention \<facility\>\<instrument\>\<filter\>

The allowed filters are:

- 2MASS\_H
- 2MASS\_J
- 2MASS\_Ks
- GALEX\_FUV
- GALEX\_NUV
- GROUND\_JOHNSON\_B
- GROUND\_JOHNSON\_U
- GROUND\_JOHNSON\_V
- GROUND\_COUSINS\_I
- GROUND\_COUSINS\_R
- TYCHO\_B\_MvB
- TYCHO\_V\_MvB
- STROMGREN\_b
- STROMGREN\_u
- STROMGREN\_v
- STROMGREN\_y
- GaiaDR2v2\_BP
- GaiaDR2v2\_G
- GaiaDR2v2\_RP
- PS1\_g
- PS1\_i
- PS1\_r
- PS1\_w
- PS1\_y
- PS1\_z
- SDSS\_g
- SDSS\_i
- SDSS\_r
- SDSS\_u
- SDSS\_z
- SkyMapper\_u
- SkyMapper\_v
- SkyMapper\_g
- SkyMapper\_r
- SkyMapper\_i
- SkyMapper\_z
- SPITZER\_IRAC\_36
- SPITZER\_IRAC\_45
- WISE\_RSR\_W1
- WISE\_RSR\_W2
- NGTS\_I
- TESS
- KEPLER\_Kp
